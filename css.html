Wstęp

CSS, a raczej Cascading Style Sheets (Kaskadowe arkusze stylów), są nieodłącznym kompanem HTML'a. Język ten służy do opisu formy prezentacji stron WWW, czyli po prostu jego wyglądu. CSS słada się ze zbiorów zasad służących do formatowania treści. Zbudowane są one z własności oraz jej wartości.

Dodawanie stylów

Istnieją 3 główne sposoby implementowania arkuszy stylów na naszej stronie, są to: styl nagłówkowy, styl wewnętrzny oraz styl lokalny. Styl nagłówkowy jak wskazuje jego nazwa, deklarujemy w sekcji <head>.

KOD
SKOPIUJ
< head >   < style >     // Tutaj wstawiamy nasze reguły CSS   < /style > < /head >

Styl zewnętrznego, tutaj musimy stworzyć oddzielny plik z rozszerzeniem .css najczęściej nazywamy taki plik jako style.css. Aby podłączyć zewnętrzny arkusz musimy skorzystać z następującego znacznika.

KOD
SKOPIUJ
< head > < link  rel= ” stylesheet ”   href= ” ścieżka do pliku ”   type= ” text/css ”> < /head >

Styl lokalny to styl który deklarujemy wewnątrz elementu HTML.

KOD
SKOPIUJ
< body > < span  style= ” color:red ” > Ten tekst będzie czerwony   < /span > < /body >
LIVE
Ten tekst będzie czerwony

Selektory CSS

Reguły CSS nie zadziałają dopóki nie przypiszemy ich do określonych elementów html, które aktualnie chcemy sformatować. Za powiązanie reguł CSS z elementami języka Html odpowiedzialne są tzw. selektory. Jest wiele sposobów tworzenia selektorów w języku CSS. Posłużą do tego m.in same elementy html oraz atrybuty id i class Selektorem może być: element html np: span, id elementu html np: #footer oraz klasa elementu np: .klasa Wszystkie reguły zapisujemy w nawiasach klamrowych które umieszczamy za naszym selektorem.

KOD
SKOPIUJ
< style >   span  {   Tutaj zapisujemy swoje reguły  }   #footer  {   Tutaj zapisujemy swoje reguły  }   .klasa  {   Tutaj zapisujemy swoje reguły  } < /style >

Reguły CSS

Na całą regułę CSS składają się 3 czynniki: selektor, własność i wartość. Przykładowa reguła która elementowi blokowemu o klasie .block nada wielkość 100px na 100px, zmieni kolor tła na zielony, obramowanie na kolor czarny o wielkości 1px, ustawi czcionkę Arial oraz nada jej wielkość 20px i kolor biały.

KOD
SKOPIUJ
< style >   .block  {  
height:100px;
width:100px;
background-color:green;
border: 1px solid black;
font-family: Arial
font-size: 20px;
color: white;  }
  < /style >
