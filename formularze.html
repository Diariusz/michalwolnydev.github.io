Wstęp

Formularze to taki element HTML'a bez którego większe witryny praktycznie nie funkcjonują. Potrzebujemy ich jeżeli chcemy się gdzieś zalogować, zarejestrować czy chociażby wysłać jakąś wiadomość prosto ze strony. Warto podczas nauki zwrócić uwagę na ten element.

Formularz

Formularz HTML tworzymy poprzez zastosowanie elementu <form>. Stworzymy teraz prosty przykładowy formularz rejestracyjny, w którym trzeba będzie uzupełnić login i hasło. Żeby wysłać nasz formularz musimy wstawić także przycisk oraz go nazwać, za nazwę odpowiada znany nam już atrybut value, <input type="submit" value="Ok">

KOD
SKOPIUJ
< form > Login  < input   type= " text " > Hasło  < input   type= " password " > < input   type= " submit "  value= " Ok " > < /form >
LIVE


W celu wysłania danych naszym formularzem html, musielibyśmy stworzyć plik z instrukcjami np. w języku PHP i zapisać go w folderze z naszą stroną internetową np. pod nazwą form.php. Do elementu <form> dodalibyśmy następnie atrybut action, który wskazywałby ścieżkę dostępu do naszego pliku form.php. Po naciśnięciu klawisza submit akcja z pliku form.php zostałaby wykonana.

KOD
SKOPIUJ
< form   action= " form.php "> Login  < input   type= " text " > Hasło  < input   type= " password " > < input   type= " submit "  value= " Ok " > < /form >

Input

Najbardziej popularnym elementem html pozwalającym na wprowadzanie danych przez użytkownika jest element <input>. Element <input> jest samozamykającym się elementem. Jednak prawidłowy jego zapis wymaga jeszcze deklaracji typu. Typu deklaracji dokonujemy poprzez atrybut type Istnieje wiele różnych typów elementu <input>, jednak najbardziej popularnym typem jest text.

KOD
SKOPIUJ
< input  type= ” text ” >
LIVE

Tekst wpisywany przez nas w elemencie <input> jest wartością tego elementu. Dane przez nas wpisane, czyli ta wartość jest przesyłana dalej formularzem. Np. gdy podamy swój adres e-mail to mówimy że wartością pola <input> jest ten adres. Wartość możemy także określić "na szytwno" stosując atrybut value

KOD
SKOPIUJ
< input  type= ” text ”  value= ” Wartość podana przez nas ” >
LIVE

Input type radio

Kolejnym dość istotnym elementem <input> jest radio, typ ten wykorzystywany jest do formularzy jednokrotnego wyboru. Żeby to zadziałało nalezy w każdym elmencie umieścić atrybut name o tej samej nazwie.

KOD
SKOPIUJ
< input  type= ” radio ”  name= ” test ” > < input  type= ” radio ”  name= ” test ” > < input  type= ” radio ”  name= ” test ” >
LIVE


W typie radio także bardzo ważnym jest wcześniej wspomniany atrybut value. W porównaniu do typu text tutaj nie podajemy sami wartości którą ma wysłać formularz tylko ją zaznaczamy. Efektem tego jest że musimy sami w kodzie podać wartość. Najczęściej podajemy to co jest wyświetlane obok pola wyboru.

KOD
SKOPIUJ
< input  type= ” radio ”  name= ” test ”  value= ” Sport ” >Sport < input  type= ” radio ”  name= ” test ”  value= ” Muzyka ” >Muzyka < input  type= ” radio ”  name= ” test ”  value= ” Motoryzacja ” >Motoryzacja
LIVE

Input type checkbox

Typ checkbox w porównaniu do radio wykorzystywany jest do wielokrotnego wyboru. Bardzo często jest także wykorzystywany do potwierdzania, np. moment w którym akceptujemy jakiś regulamin strony.

KOD
SKOPIUJ
< input  type= ” checkbox ” >Sport < input  type= ” checkbox ” >Muzyka < input  type= ” checkbox ” >Motoryzacja
LIVE
Sport
Muzyka
Motoryzacja

W typie checkbox nie korzysta się z atrybutu value, zamiast tego używa się atrybutu checked. Jeżeli dany element zawiera atrybut checked to jest on wysyłany dalej poprzez formularz.

KOD
SKOPIUJ
< input  type= ” checkbox ”  checked >Sport < input  type= ” checkbox ” >Muzyka < input  type= ” checkbox ” >Motoryzacja
LIVE
Sport
Muzyka
Motoryzacja

Textarea

Jeżeli chcemy wprowadzić więcej danych niż jedno, dwa słowa to zamiast elementu <input> lepiej jest użyć <textarea>. Textarea to wieloliniowe pole które pozwala nam na wpisywanie nie tylko krótkich form pisemnych, ale także długie wypracowania.

KOD
SKOPIUJ
< textarea > </textarea>
LIVE

Aby pobrać wpisaną przez nas wartość z pola <textarea> musimy nadać atrybut name o wybranej przez nas nazwie.

KOD
SKOPIUJ
< textarea   name= ” naszaNazwa ” > < /textarea >

Select

<Select> jest elementem który pozwala nam na utworzenie listy rozwijanej. Przy tym elemencie musimy także korzystać z <option> dzięki któremu wstawimy do naszej listy opcje do wyboru. Element <select> wymaga użycia atrybutu value jeżeli chcemy wysłać wartości formularzem.

KOD
SKOPIUJ
< select > < option   value= ” opcja1 ” > < /option > < option   value= ” opcja2 ” > < /option > < /select >
